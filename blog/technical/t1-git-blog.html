What is Version Control?

Version control allows you to save your progress on your work over time so you can look back at what you did and revert back to those changes at a later date. This is important when working on live software products because if you release a new version and something breaks, you can just revert back to the previous release. It is like the "Save Game" functionality in a video game.

What is Git?

Git is a version control system that allows programmers collaborate on team projects, show off what they have done, get help from other programmers around the world, and (obviously) track changes to their projects. Git works with GitHub, a free-to-use, cloud-based, version control service.

How to use GitHub

GitHub can be tricky to use. Let's just make a repository, clone it (save it) to your computer, and celebrate.
1) Go to GitHub.com and make a free account.
2) Click on the "Repositories" tab.
3) Click "New" (green button, top right of screen).
4) Name your repository something relevant to your project
5) Copy the URL displayed at the top of the page.
6) Open Terminal (your command line interface)
7) Navigate to the folder you want your local GitHub repository to live in (a local copy of your online GitHub repo, used for making changes before showing them to the world)
8) Initialize your repo - type: git init
9) Add a readme - type: git add README.md
10) Commit your changes (committing is like adding your changes to your GitHub repo with a comment on what you did) - type: git commit -m "add readme"
11) Add a remote for your changes (don't worry about understanding this for now) - type: git remote add origin [COPIED URL]
12) Add your changes to your online GitHub repo - type: git push -u origin master
13) Celebrate!

Congratulations! Let's review what you did. You made a GitHub account, created an online repository, cloned (copied) your online repo onto your computer, made a change to your local repo on your computer, and added (push) that change back to your online repository. You have made history.
