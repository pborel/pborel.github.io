T4 - Enumerables

5/30/15

According to the Bastard's Book of Ruby, "The Enumerable module provides a set of methods to traverse, search, sort and manipulate collections." This is a fancy way of saying that enumerable methods are methods that allow easy looping through arrays and hashes easily. They are short cuts were made to be easier to write and read. The best way to understand enumerable methods is through example:

For printing out each element in an array you can use a for loop:

ark = ['cat', 'dog', 'pig', 'goat'] for animal in ark puts ark[animal] end


Alternatively, you can utilize the enumerable each method:

ark = ['cat', 'dog', 'pig', 'goat'] ark.each do |animal| puts animal end

Or:

['cat', 'dog', 'pig', 'goat'].each{ |animal| puts animal }
As you can see, the enumerable each method lets us make code that is much cleaner and tighter.
